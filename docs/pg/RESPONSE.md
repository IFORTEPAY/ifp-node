# Response code & message

Here is the list of possible responseCode and responseMessage for PG API.

| Response Code | Response Message        |
| ------------- | ----------------------- |
| 00            | Success                 |
| 01            | Request Parameter Error |
| 02            | Data Not Found          |
| 03            | Failed Request          |
| 04            | Unauthorized Merchant   |
| 98            | System Error            |
| 99            | Unknown Error           |
